<<概要>>
　全国高校生プログラミングコンテストで使用されたCHaserという競技用の、
GUI操作クライアントアプリです。
（参考：http://www.zenjouken.com/?page_id=517）

　この競技は通常、毎ターンの行動を自動で決定するクライアントの
アルゴリズムを競うものですが、このアプリケーションは
毎ターンの操作を画面上で選択して確定するものです。

　なんら公式なものではありませんので、ルール細部やプロトコル等について
実際の競技と異なる可能性がありますので、参考程度にご利用ください。


<<動作条件>>
　基本的に.NET Framework 4.0が動く環境であれば動くはずなので、
Windows XP, Vista, 7, 8, 8.1等で実行できるだろうと思います。

　不具合については、諦めていただくか、下部記載の連絡先までお知らせください。


<<ファイル構成>>

	CHaserGuiClient > CHaserGuiClient.exe			//実行ファイル
			 		> Oika.Libs.MeLogg.dll			//ログ出力用ライブラリファイル
			 		> CHaserGuiClient.exe.config	//.NET Framework設定ファイル
			 		> readme.txt					//READMEファイル（このファイル）


<<インストール>>
　CHaserGuiClient.zipを解凍し、適当な場所に置いてください。


<<アンインストール>>
　CHaserGuiClientのフォルダを丸ごと削除してください。


<<遊び方>>
●ソフトの起動
　CHaserGuiClient.exeをダブルクリックで実行してください。
上部にチーム名と接続先情報を入力し、Startをクリックしてください。
各ターンは、GetReady→メソッド実行の順に実行してください。
